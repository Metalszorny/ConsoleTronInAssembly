# ConsoleTronInAssembly


About:
------
A tron game in drawing mode. Two lines, blue and red race agains each other and tries to make the other player crash.


Used Language:
--------------
- Assembly


Features:
---------
- Drawing Mode
- User Input


Enviroment:
-----------
- Development: Notepad++
- OS: Microsoft Windows XP Service Pack 3, 32bit
- Test: Command Prompt
